# 🧠 Breast Cancer Detection AI

## 🔍 Overview

This project is a Machine Learning-based system that predicts whether a tumor is **benign or malignant** using medical data and image processing techniques.

The system combines multiple approaches including **Support Vector Machine (SVM)**, **Convolutional Neural Networks (CNN)**, and **Principal Component Analysis (PCA)** for feature extraction and classification.

---

## ⚙️ Tech Stack

* Python
* Machine Learning (SVM, CNN)
* NumPy, Pandas, Scikit-learn
* Image Processing
* HTML, CSS, JavaScript (Frontend)

---

## 🚀 Features

* Data preprocessing and cleaning
* Feature extraction using Gabor filters
* PCA for dimensionality reduction
* SVM-based classification
* CNN model for image-based prediction
* Frontend interface for user interaction
* Modular and scalable code structure

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/Mohamed-shihaab/breast-cancer-detection-ai.git
cd breast-cancer-detection-ai
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the backend

```
python backend/app.py
```

### 4. Open the frontend

Open the `frontend/index.html` file in your browser.
