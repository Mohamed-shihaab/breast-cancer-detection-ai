# Breast Cancer Prediction System

## 🔍 Overview
This project uses Machine Learning to predict whether a tumor is benign or malignant based on medical data.

## ⚙️ Tech Stack
- Python
- Machine Learning (SVM / CNN)
- NumPy, Pandas, Scikit-learn

## 🚀 Features
- Data preprocessing
- Feature extraction
- Model training and prediction
- Image-based prediction (if applicable)

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python app.py
